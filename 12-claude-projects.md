# Claude Projects — Organização de Trabalho Recorrente

Claude Projects é uma funcionalidade poderosa para organizar trabalho recorrente com contexto persistente.

---

## Para que serve

Claude Projects permite organizar conversas, arquivos e contextos em espaços de trabalho separados, mantendo uma base de conhecimento e padrão de resposta para cada tema. Em vez de repetir informações a cada chat, o projeto mantém tudo centralizado.

---

## O que você pode fazer

### 1. Criar espaços dedicados para temas específicos

Cada Project funciona melhor quando representa um processo recorrente: criar conteúdos, revisar campanhas, analisar relatórios, responder clientes, estudar documentos ou manter o tom de voz de uma marca.

### 2. Adicionar uma base de conhecimento persistente

Dentro de um projeto, o usuário pode enviar documentos, textos, códigos ou outros arquivos. Esses materiais ajudam o Claude a entender melhor o contexto das conversas.

### 3. Definir instruções personalizadas

É possível definir instruções personalizadas, como tom de voz, formato das respostas ou até a perspectiva profissional que a IA deve adotar.

### 4. Manter histórico organizado por conversa

Múltiplas conversas dentro do mesmo projeto compartilham o contexto, mas cada uma é independente — perfeito para organizar diferentes campanhas, clientes ou fases de um trabalho.

---

## Recursos avançados (planos pagos)

Nos planos pagos (Pro, Max, Team e Enterprise), o Claude conta com **Geração Aumentada por Recuperação (RAG)**, que expande a capacidade do projeto em até dez vezes sem perda de qualidade quando o volume de informações se aproxima do limite de contexto.

---

## Quando usar

Claude Projects funciona melhor quando o trabalho se desdobra ao longo do tempo e ganha força a cada iteração. Para trabalhos pontuais, uma conversa regular é suficiente.

### Exemplos práticos

| Caso de uso | Benefício |
|---|---|
| Briefs de conteúdo | Redução de até 80% do tempo dedicado |
| Documentação e FAQs | Contexto acumulado entre sessões |
| Copywriting para produtos/e-mails | Tom de voz consistente |
| Análise de relatórios | Base de dados centralizada |
| Atendimento ao cliente | Padrão de resposta estável |
